To disable RTCPeerConnection and protect IP addresses leakage, go to 
about:config ( type in url bar )
and toggle 
media.peerconnection.enabled to false.

To disable Media Devices, toggle 
media.navigator.enabled to false
also
media.peerconnection.enabled to false
